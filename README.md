##  Project Title
**User Funnel & Revenue Performance Analysis using SQL and Power BI**

##  Objective

This project is the final stage of the internship.  
The goal is to analyze user behavior in a digital platform using real-world data.

In this project, I worked on:

  - User behavior analysis using SQL
  - Funnel analysis to identify drop-off points
  - Revenue and performance analysis
  - Interactive dashboard creation using Power BI
  - Business insights and recommendations



##  Business Problem

A digital company wants to understand:

  - How users move through the platform
  - Where users drop off in the funnel
  - Which marketing channels perform best
  - Which devices generate higher engagement
  - Which regions generate more revenue
